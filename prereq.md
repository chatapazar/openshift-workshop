## Install Operator
- Web Terminal
- AMQ Streams
- Serverless (create knative-serving in project knative-serving)
- ElasticSearch
- ClusterLogging
  - required 3 node of worker
  - create clusterlogging instance
    ```yaml
    apiVersion: "logging.openshift.io/v1"
    kind: "ClusterLogging"
    metadata:
      name: "instance" 
      namespace: "openshift-logging"
    spec:
      managementState: "Managed"  
      logStore:
        type: "elasticsearch"  
        retentionPolicy: 
          application:
            maxAge: 3d
          infra:
            maxAge: 3d
          audit:
            maxAge: 3d
        elasticsearch:
          nodeCount: 3 
          storage:
            storageClassName: gp2
            size: 200G
          resources: 
            requests:
              memory: "8Gi"
          proxy: 
            resources:
              limits:
                memory: 256Mi
              requests:
                memory: 256Mi
          redundancyPolicy: "SingleRedundancy"
      visualization:
        type: "kibana"  
        kibana:
          replicas: 1
      collection:
        logs:
          type: "fluentd"  
          fluentd: {}
    ```
  - create index infra and app in kibana
